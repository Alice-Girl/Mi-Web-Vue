> 数据状态管理 `storage` 封装

- `Cookie` 、`localStorage` 、`sessionStorage` 三者区别?

    - 存储大小： `Cookie` 4K 、`Storage` 5M
    - 有效期：`Cookie` 拥有有效期 、`Storage` 
    - `Cookie` 会发送到服务器端，存储在内存中 、 `Storage` 只存储在浏览器端
    - 路径：`Cookie` 有路径限制 、 `Storage` 只存储在域名下
    - API：`Cookie` 没有特定的 **API** 、`Storage` 有对应的 **API**
